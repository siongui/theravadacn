佛陀曾传授四种崇高的心灵境界:

慈爱 (慈， metta )
同情 (悲， karuna )
随喜 (喜， mudita )
舍离 (舍， upekkha )

巴利文佛教经典中，此四种心境称为"Brahma-vihara"[梵住、梵天之居处]。这个词既可表达为: 卓越、崇高、升华的心境，又可表达为: 梵天般的、天神般的、神圣的居处。
